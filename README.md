This project is an implementation of the **Ludo** board game, developed in **JavaScript**, using **HTML, CSS, and images** to represent the board, dice, and pieces.

The game allows users to simulate an interactive Ludo match directly in the web browser.

---

## Features

- Visual Ludo board
- Game pieces in different colors (blue, red, green, yellow)
- Dice rolling
- Piece movement according to the game rules
- Graphical interface with images
- Runs directly in the browser

---

## Technologies Used

- **HTML**
- **CSS**
- **JavaScript**
- Images (`.jpg`, `.png`, `.gif`)
- (Optional) 3D Models / Three.js

---

## How to Run the Game

1. Download the project or clone the repository
2. Open the `index.html` file in your web browser  
   (Chrome, Edge, or Firefox)
3. The game will start automatically
